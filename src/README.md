# Source Code Overview

This directory contains the core code for ArbiLoop.

## Components
1. `frontend/`: Next.js dashboard and execution UI.
2. `contracts/`: Solidity contracts for execution logic.
3. `supabase/`: SQL schema for user settings.
4. `bot/`: Telegram bot backend for wallet linking and alert settings.
